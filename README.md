# Pelican Chem Website

Modern enterprise-grade corporate website for **Pelican Chem** — manufacturer and supplier of RO chemicals, antiscalants, boiler chemicals, ETP chemicals, and water treatment solutions.

Built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for Production

```bash
npm run build
npm start
```

## Site Architecture

```
/                          Homepage
/about                     About Us (company profile, vision, mission, timeline)
/products                  Product catalog
/products/[category]       Product category pages
/products/[category]/[slug] Individual product pages
/industries                Industries overview
/industries/[slug]         Industry detail pages
/custom-label              Own Label RO Chemicals
/quality                   Quality & Manufacturing
/contact                   Contact page with form and map
```

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **SEO:** Metadata API, sitemap.xml, robots.txt

## Design

Premium industrial branding inspired by modern enterprise web design:

- Deep Navy Blue, Aqua Blue, Teal accent palette
- Full-width sections with generous whitespace
- Scroll-triggered fade-in animations
- Animated counters and hover interactions
- Mobile-first responsive layout
